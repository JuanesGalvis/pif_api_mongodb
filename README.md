## Description

API para almacenar y consultar registros en una base de datos (MongoDB):
- `GET: /registro` -> Consultar todos los registros
- `POST: /registro` -> Guardar un nuevo registro en la base de datos

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
